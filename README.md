## Install

Download the **phi.zsh-theme** and place it inside the `~/.oh-my-zsh/themes` folder.

Then change the theme within the `~/.zshrc` file.

## Features

- Git branch (with alert on changes)
- Latest git commit hash
- phi color dynamic
